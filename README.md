## 📖 Sobre o Projeto

A **Clínica Dermalux** é uma landing page profissional desenvolvida para apresentar os serviços, profissionais e diferenciais de uma clínica de dermatologia. O projeto foi criado com foco em:

- **Design sofisticado** com paleta de cores em azul marinho e dourado
- **Experiência do usuário** fluida com scroll suave e navegação intuitiva
- **Responsividade** completa para todos os dispositivos
- **Componentização** para facilitar manutenção e escalabilidade
